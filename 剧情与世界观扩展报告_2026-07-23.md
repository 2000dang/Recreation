## 三、地图悬浮窗模块设计说明

### 3.1 实现位置

**文件**：`dist/V20260721/悬浮球状态栏.js`（悬浮球面板内置）

### 3.2 入口

悬浮球面板顶栏新增 🗺️ 按钮（位于主题切换 ☀️ 和刷新 🔄 之间），点击弹出模态弹窗。

### 3.3 交互逻辑

1. **点击 🗺️** → 读取 `stat_data.环境与系统` 中的 `当前楼层`/`当前位置`/`当前场景`
2. **在公司内部** → 渲染 25 层写字楼全楼层列表（B2-28F），当前楼层以绿色高亮 + 左边框标记"← 当前位置"
3. **在公司外部** → 显示 📍 当前位置标识 + 外部场景文本 + 提示"进入环晓科技大厦后显示公司内部楼层地图"
4. **每层标注**：楼层号 + 功能名称 + 图标（🔒机密层/🚫限制层/🏥医疗/🍽️餐饮/🅿️停车场/🏢办公）
5. **当前详情**：弹窗底部显示当前楼层/位置/场景三行数据

### 3.4 UI布局

```
┌─────────────────────────────────┐
│  📍 环晓科技总部 · 25层写字楼   │
├─────────────────────────────────┤
│ B2  🅿️ 地下停车场               │
│ B1  🔒 数据中心·Sisterhood机密   │
│ 1F  🏢 大厅前台·门禁            │
│ 4F  🏢 待分配中心·面试间        │
│ 6F  🍽️ 公共食堂·社交中心        │
│ 8F  🏢 研发部·天台吸烟区 ← 当前 │  ← 绿色高亮
│ 9F  🚫 女员工专用层(泳池/健身)   │
│ ...                             │
│ 28F 🔒 人体实验室·姜舞主场      │
├─────────────────────────────────┤
│ 楼层：8层·研发部  位置：个人办公区 │
│ 场景：办公室                    │
└─────────────────────────────────┘
```

### 3.5 数据源

- `stat_data.环境与系统.当前楼层` — 地图高亮依据
- `stat_data.环境与系统.当前位置` — 详情面板
- `stat_data.环境与系统.当前场景` — 详情面板

### 3.6 15层地图覆盖
